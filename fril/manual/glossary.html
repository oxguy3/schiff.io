Fril - Glossary of terms used in built-in predicate reference manual

All material © Fril Systems Ltd 1999

asc_int

an integer between 0 and 252, corresponding to an ASCII character code (0 - 127), or an extended character code

atom

a list.

atom_list

a list of atoms.

body

a list of atoms.

char

a Fril constant whose print-name is a single character.

cl_supp

either a support or a list of two supports.

clause

a Fril clause, i.e. a list whose first element is a list with a constant predicate name as first element. The most general form for is ((<const> | ) |)

const

A Fril constant.

dtype

a Fril dtype (discrete fuzzy set).

element

evidential weight

filename

a Fril constant which names a file.

float

a floating point decimal number. Single precision arithmetic is used in Fril, so rounding errors may occur.

fset

a Fril itype (fuzzy number), dtype (discrete fuzzy set), or universe.

goal

goal_list

in_stream

a Fril constant which names a currently open input stream.

int

an integer.

io_stream

a Fril constant which names a stream currently open for input OR output.

itype

a Fril itype (fuzzy number), e.g. [0:0 1:1 2:1 3:0].

list

named_fset

nni

a non-negative integer.

num

a Fril number, either float or int, e.g. 1, 2, 3.2, 1.34e-7

out_stream

a Fril constant which names a currently open output stream.

probability

a number in the range 0 to 1.

support

either a number in the range 0 to 1, or a list of two numbers in the range 0 to 1. In the latter case, the first number must be less than or equal to the second, e.g. 0.6 or (0.8 1).

term

any valid Fril object, i.e. a list, number, constant, variable or fuzzy set.

univ

a constant naming a Fril universe.

var

a Fril variable.

var_support

either a variable, or a list of two variables, e.g. S or (L U).
